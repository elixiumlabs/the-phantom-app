## 📊 Product Overview

### What It Does
Phantom helps founders validate brand ideas before launch through a structured 4-phase process:

1. **Phase 01 - Ghost Identity:** Define problem, positioning, and voice using AI
2. **Phase 02 - Silent Test:** Build minimum offer, run outreach, track conversions
3. **Phase 03 - Iteration Loop:** Diagnose issues, iterate based on data
4. **Phase 04 - Lock In:** Finalize brand identity from validated buyer language

### Key Features
- **17 AI Generators** powered by Google Gemini (problem refinement, positioning, offer building, objection analysis, etc.)
- **Proof Vault** for storing testimonials, screenshots, case studies, revenue data
- **Signal Tracker** for cross-project outreach analytics
- **Outreach Templates** library with 8 proven frameworks
- **Real-time Collaboration** via Firestore sync
- **Stripe Billing** with monthly/annual subscriptions
- **PDF Export** of final brand lock-in guide
